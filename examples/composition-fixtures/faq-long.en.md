::::faq{title="Operations and traceability" role="primary"}
:::question{title="How can a published composition remain technically and editorially traceable years after it was created?" category="Governance"}
ContentKit preserves the authored Markdown, normalized semantics, resolved composition, structured diagnostics, and reproducible hashes. Operators can verify the original claim, the selected information pattern, and every responsive fallback without relying on an executable design payload.
:::
:::question{title="What happens when an embedded content region is substantially narrower than the browser viewport?" category="Responsive design"}
Pattern resolution treats the actual container width as the effective boundary. Text reflows first, structures stack second, and a bounded static export can reduce visible evidence only when it reports the omission through a machine-readable diagnostic.
:::
:::question{title="How does the agent contract prevent an attractive visual form from changing the intended meaning?" category="Agents"}
Every pattern declares implications, rejection reasons, slots, capabilities, budgets, and static fallbacks. Agents can express a preference but cannot inject coordinates, executable layout code, CSS classes, or renderer settings.
:::
