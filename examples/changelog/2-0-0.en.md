# Atlas 2.0

## Added

- Versioned documentation with hierarchical navigation.
- Personal reader accounts and group policies.
- Protected search and navigation projections.

## Changed

Access rules become active with the release. A sitemap or search index can
therefore never switch before or after its corresponding protection rule.

## Security

Passwords use `scrypt` with a unique salt. The database stores session tokens
only as HMAC hashes.
